# /speckit.constitution Command

**Purpose**: Create or update project governing principles and development guidelines.

**When to use**: At the start of a project or when updating project principles.

## Instructions

1. Create or update `.specify/memory/constitution.md`
2. Define project principles including:
   - Purpose & Scope
   - Technology Philosophy
   - Code Quality Standards
   - Architecture Guidelines
   - User Experience standards
   - Performance & Scale requirements
   - Security & Safety guidelines
   - Development Process
   - Complexity tracking rules
3. All subsequent development should align with these principles

## Output

Creates/Updates: `.specify/memory/constitution.md`
