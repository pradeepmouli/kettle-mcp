- `package.json` - Node.js/TypeScript dependencies and scripts
- `tsconfig.json` - TypeScript compiler configuration
- `.eslintrc.json` - ESLint configuration
- `jest.config.json` - Jest test configuration

The original Python specification has been preserved in `.specify/memory/constitution.md` 
with updates to reflect the TypeScript implementation.
